UPOS в.11.41.03 для торговых автоматов


ВАЖНО:
~~~~~~

1. Для установки скопируйте все содержимое каталога Files на жесткий диск
   торгового автомата в каталог, указанный в инструкции к основному ПО автомата.

2. В файле haldrv.ini укажите номера СОМ-портов, к которым подключены картридер
   и пинпад.

3. - Если используется пинпад SZZT 598 или 588 и картридер Sankyo, то скопируйте файл 
   HalCfg_Szzt_Sankyo.dat в файл HalCfg.dat. 
   
   - Если используется пинпад Sagem1217 и картридер Sankyo, то скопируйте файл 
   Halcfg_Cryptera_Sankyo.dat в файл HalCfg.dat.

   - Если используется пинпад SZZT 598 или 588 и картридер Neuron MSA-790U-3R, то скопируйте файл 
   Halcfg_Szzt_Neuron.dat в файл HalCfg.dat.

   - Если используется пинпад Sagem1217/1215 и картридер Neuron MSA-790U-3R, то 
   оставьте файл HalCfg.dat без изменений.

4. Запустите файл UpConf.exe.

5. В окне программы укажите текст заголовка чека, номер терминала, 
   номер мерчанта, и нажмите кнопку "Создать настроечный файл". 
   Должно появиться сообщение "Файл r004.r успешно создан".

6. Запустите программу LoadParm.exe. При старте должны автоматически 
   прогрузиться настроечные параметры, и файл r004.r удалится. Программа готова
   к работе.

   НИ В КОЕМ СЛУЧАЕ НЕЛЬЗЯ СОЗДАВАТЬ *.tlv-ФАЙЛ "С НУЛЯ" ИЛИ БРАТЬ ЗА ОБРАЗЕЦ
   НАСТРОЙКИ ОТ ОБЫЧНЫХ POS-ТЕРМИНАЛОВ !!!

7. Подготовка пинпадов к работе описана в каталоге Doc. Чтобы правильно заказать 
   и потом ввести ключи в пинпад, необходимо иметь ввиду следующее:

  - Пинпады SZZT используют ключи KLK, как обычные POS-терминалы. С помощью
    программы LoadParm.exe в меню "Служебные операции\Настройки\Техобслуживание\
    \Ключи\Состояние" нужно посмотреть серийный номер пинпада, заказать ключ
    KLK для этого серийного номера, а потом ввести его с помощью меню
    "Служебные операции\Настройки\Техобслуживание\Ключи\Ввод\KLK".

  - Пинпады Sagem 1217 не умеют работать с ключом KLK. Для таких пинпадов 
    необходимо заказывать МАС- и PIN-ключ с привязкой к регистрационному 
    номеру терминала (как для ИПТ).
